# Local Government Data Privacy Simulation

This example simulates how different 'local tools' or departments within a government might access citizen data without a centralized privacy framework. It demonstrates the risks of over-privilege and lack of granular control, where sensitive information can be exposed unnecessarily across disparate systems, illustrating the 'zero security barrier' problem.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
